# 🚀 Surcagua Studio - Portfolio de Apps

Sitio web oficial de Surcagua Studio, desarrolladores de aplicaciones móviles innovadoras para iOS.

🌐 **Website:** [www.surcaguastudio.com](https://www.surcaguastudio.com)  
📧 **Soporte:** support@surcaguastudio.com

---

## 📱 Aplicaciones

### 💰 Budgely
Control de presupuesto personal con análisis diario, mensual y anual mediante gráficos intuitivos.

### 🔒 SilentVault
Bóveda privada para ocultar y proteger fotos y videos con encriptación de nivel militar.

### 🎸 Soundforia
Suite completa para compositores y bandas: crea canciones con acordes, letras y metrónomo profesional.

### 💕 Tuyyo
La app definitiva para parejas: chat privado, álbum compartido, aniversario, mapas, juegos y más.

## 🛠️ Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Diseño moderno con variables CSS y animaciones
- **JavaScript Vanilla** - Sin dependencias, código limpio
- **Cloudflare Workers** - Hosting con Static Assets (2026)
- **Cloudflare Email Routing** - Email profesional gratuito
- **FormSubmit** - Envío de formularios sin backend
- **Git Integration** - Deploy automático desde GitHub

## 🚀 Despliegue

Este sitio está desplegado en **Cloudflare Workers con Static Assets** (2026).

**Deploy automático:** Cada push a la rama `main` despliega automáticamente a producción en 1-2 minutos.

**Servicios configurados:**
- ✅ Hosting: Cloudflare Workers (gratis)
- ✅ Email: Cloudflare Email Routing (gratis)
- ✅ Formulario: FormSubmit (gratis e ilimitado)
- ✅ SSL/TLS: Automático (gratis)
- ✅ CDN: Global (gratis)

📖 **Guías completas:**
